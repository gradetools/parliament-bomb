# Parliament Bomb

simple discord bot

## Features

- Logging
- Weekly status report (TBA)
- Moderation
- Role hierarchy management (TBA)
- Easy-to-use API

## building
run these commands, in this order:
(special thanks to [bolt926](https://github.com/bolt926))

```
aclocal
autoheader
autoconf
automake --add-missing
./configure
./config.status
make (make target)
```
